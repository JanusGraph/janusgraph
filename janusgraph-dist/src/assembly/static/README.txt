Janus: Distributed Graph Database

Janus is a distributed graph database optimized for storing
and processing large-scale graphs within a multi-machine cluster.

Janus documentation can be found online at:

  http://s3.thinkaurelius.com/docs/janus/current/

Janus is provided by Aurelius [http://thinkaurelius.com]
"Applying Network Science and Graph Theory"
